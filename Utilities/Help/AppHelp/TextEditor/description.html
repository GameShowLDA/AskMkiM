Описание работы редактора

Редактор предназначен для отображения и редактирования текстов с поддержкой:

  • Подсветка синтаксиса — цветовое выделение кода для лучшей читаемости.
  • Перенос строк — автоматический перенос длинных строк.
  • Нумерация строк — отображение номеров строк слева от текста.
  • Drag-and-Drop — перетаскивание текста с помощью мыши.

Нажмите F1 в приложении, чтобы открыть этот раздел автоматически.
