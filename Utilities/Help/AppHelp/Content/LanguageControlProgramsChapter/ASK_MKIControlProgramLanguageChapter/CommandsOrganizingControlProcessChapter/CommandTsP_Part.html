5.1.5 Команда ЦП (Цикл Повторения)

ЦП — повторно выполняет указанный участок программы.

Синтаксис

<номер> ЦП [Д] <метка-начало>[-<метка-конец>] [(кол-во)]
  • метка-начало и метка-конец — границы участка.
  • Если указана одна метка — повторяется только эта команда.
  • По умолчанию количество циклов = 1.
  • Ключ Д (документирование) выводит итоговый отчёт в протокол.

Правила

  • Участок должен находиться в тексте ПК выше команды ЦП.
  • Метка начала ≤ метки конца.
  • Для парных команд (КН/КИ, ЦГ/ПТ, ОТ/ГИ) обе команды должны лежать внутри повторяемого участка.

Примеры

190 ЦП 110-160(5)    — повторить команды 110…160 пять раз
200 ЦП 170           — повторить только команду 170
210 ЦП Д,180-180     — задокументировать повтор команды 180
220 ЦП 200(2),Д      — повторить команду 200 дважды и задокументировать
