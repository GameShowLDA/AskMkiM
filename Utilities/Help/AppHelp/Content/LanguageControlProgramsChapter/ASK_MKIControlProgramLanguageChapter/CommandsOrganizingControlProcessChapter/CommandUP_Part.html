5.1.7 Команда УП (Условный Переход)

Все команды ПК по умолчанию выставляют признак «Норма». При ошибке они после завершения меняют признак на «Брак».

Команда УП проверяет признак предыдущей команды:

  • Если признак «Брак», выполняется переход к команде с указанной меткой.
  • Иначе продолжается выполнение следующей строки.
Явный переход «сама на себя» запрещён. Перед переходом обрабатывается нажатие Esc для остановки ПК.

Синтаксис

<метка> УП <метка-цели>

Пример

200 ПР *ССИРТ*
210 УП 1000   // если «Брак», перейти на 1000
... 
1000 КЦ      // конец программы
