5.3.7 Команда ГИ — генератор импульсных коммутаций

Команда ГИ позволяет задать во времени последовательность кратковременных подключений/отключений точек ОК к шинам с помощью вложенных команд ПТ и ОТ. Используется для формирования импульсных диаграмм, циклических воздействий и временно-точных последовательностей коммутации.

Общий формат

МЕТКА ГИ
( 
   ПТ ...    | ОТ ...
   ПТ ...    | ОТ ...
   ...
)
  • МЕТКА — номер/идентификатор строки ПК.
  • В скобках перечисляются вложенные команды ПТ/ОТ в порядке выполнения.
  • Каждая вложенная команда занимает отдельную строку.

Отличия вложенных ПТ/ОТ внутри ГИ

  • Перед мнемоникой вложенной команды метка не указывается.
  • Отступы (пробелы) перед вложенными строками допускаются и рекомендуются для читабельности.
  • Остальной синтаксис параметров (время, перечисление шин и точек, ключи) как в обычных ПТ / ОТ.

Тайминг

Вложенные команды исполняются последовательно. Указанное во вложенной строке время (например, 100мс) трактуется как длительность этапа до перехода к следующей строке. Таким образом строится временная диаграмма. Имейте в виду задержки на передачу команд и срабатывание реле — фактические фронты подключения смещены относительно номинальных «моментов привязки».

Простой пример (импульсная последовательность)

60 ГИ
(
  ПТ 10мс              // пауза 0–10 мс
  ПТ 100мс *B2:T1*     // подключить T1 к B2 на 10–110 мс
  ПТ 100мс             // пауза 110–210 мс
  ПТ 100мс *B2:T2*     // импульс T2 к B2 на 210–310 мс
  ПТ 100мс             // пауза 310–410 мс
  ПТ 100мс *A2:T3,T4*  // импульс двух точек на 410–510 мс
)

Пример с запуском по событию и смешанными действиями

ГИ часто используют в связке с командами контроля/запуска (КН, КИ, ЦГ). Во вложенных ПТ / ОТ можно применять ключи П (ждать «Пуск») и С (ждать «Стоп»). 

50 КИ  T=10мс+-5, ПА1<5В, СA1<5В, ШB1, 600мс

60 ГИ
(
  ПТ *B2:1,3*     // подключить точки 1 и 3 к B2
  ПТ 30мс         // выдержка
  ОТ *B2:1*       // отключить точку 1
  ПТ 100мс        // выдержка
  ПТ 100мс *A2:2* // импульс точки 2 на A2
  ПТ С            // ждать сигнала "Стоп" (от КИ)
  ОТ *B2:3*       // отключить точку 3
  ПТ 100мс *A2:4,5* // финальный импульс двух точек
)

Советы по применению

  • Используйте ГИ, когда временная диаграмма важнее читаемости набора отдельных ПТ / ОТ.
  • Комбинируйте паузы (ПТ без точек, только время) с активными импульсами (ПТ/ОТ с точками).
  • Для событийно‑зависимых диаграмм включайте ключи П / С и управляйте запуском через КИ.
  • Помните о структуре шин: команды управления коммутациями допустимы в режиме ВШ 4Ш; при 2Ш они запрещены.
