2.3 Останов выполнения программы контроля

Во время выполнения ПК ПО АСК реагирует лишь на Esc (или щелчок левой кнопкой мыши, который дублирует Esc). После нажатия выполнение приостанавливается, а в нижней строке появляется сервисное меню:

Пункты меню

  • Y — Продолжить  — возвращает ПК к выполнению с текущей команды.
  • Останов  — завершить текущую команду и остановиться так, как если бы она была запущена одиночным шагом (F8, Run / Command).
  • O — Опции  — открывает окно Options (см. п. 5).
  • I — Инфо  — открывает окно Info (см. п. 6).
  • Esc — Прервать ПК  — немедленно прекращает выполнение, сбрасывает аппаратуру в исходное состояние. Перед прерыванием выводится дополнительный запрос‑подтверждение.

Индикация и статусная строка

После выбора любой команды меню в окне протокола (верхняя строка) кратко показывается строка статуса — атрибуты и результаты последней выполненной команды. В нижней строке указывается причина останова, например:

«Останов: по требованию оператора»

Сообщение и статусная строка скрываются при первом нажатии любой «обычной» клавиши (кроме управляющих: Ctrl, Alt, Shift, Caps Lock, Num Lock).
