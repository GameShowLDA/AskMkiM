2.1.6 Команда Run / Toggle — установка / сброс точки останова

Горячая клавиша: Ctrl‑F8

Назначение

Добавляет или снимает точку останова на команде, указанной курсором в окне «Программа». При достижении такой точки выполнение ПК автоматически приостанавливается (маркер остаётся на строке команды, фон её первой строки подсвечен красным).

Правила использования

  • Активным должно быть окно «Программа».
  • Курсор можно поставить на любую строку между началом текущей команды и началом следующей — точка будет привязана к самой команде.
  • Всего допускается до 10 остановов одновременно.
  • Положение остановов запоминается для последних 100 файлов ПК (сохраняется в конфигурации ПО АСК).

Переключение

Повторное нажатие Ctrl‑F8 на той же команде снимает ранее установленную точку останова.
Если точек уже 10, попытка добавить ещё одну вызывает звуковой сигнал и игнорируется.

Советы

  • Используйте в комплекте с F8 («Command») для пошаговой отладки длинных последовательностей.
  • После паузы можно продолжить выполнение Ctrl‑F9 (Run / Run) или выполнить одну команду F8.
