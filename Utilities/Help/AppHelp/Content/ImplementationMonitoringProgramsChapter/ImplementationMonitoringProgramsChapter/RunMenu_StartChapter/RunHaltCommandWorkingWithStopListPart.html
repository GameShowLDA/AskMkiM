2.1.7 Команда Run / Halt — работа со «Списком остановов»

Горячая клавиша: Ctrl‑F6 (по умолчанию в ПО АСК)

Назначение

Открывает вспомогательное окно «Список остановов», где отображаются все текущие точки останова, установленные в окне «Программа».

Содержимое окна

  • Каждый элемент — первая строка команды, на которой стоит останов.
  • Элементы упорядочены по меткам команд (алфавитно).
  • В нижней строке окна отображается подсказка с доступными клавишами.

Навигация

Клавиша Действие
↑ / ↓ Выбор пункта списка
Enter Перейти в окно «Программа» и перенести курсор на связанную команду (маркер выполнения не меняется)
Del Удалить выбранную точку останова

Замечания

  • Список помогает быстро ориентироваться в остановах и очищать их.
  • Удаление в списке снимает красную подсветку той же команды в окне «Программа».
  • Изменения сохраняются сразу же, даже после закрытия окна.
