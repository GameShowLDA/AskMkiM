1.2 Опции выполнения программ контроля

Параметры, влияющие на то, как выполняются программы контроля и какие сообщения выводятся на экран, собраны в меню Options → Опции (п. 5).

Установка «по умолчанию»

Чтобы быстро вернуться к стандартному (минимальному) объёму сообщений в окне «Протокол выполнения», выберите пункт:

Options → Опции → Установить по умолчанию → «Выполнение программ и тестов»

Размеры окон

Команда Ctrl + F5 (Window / Size) позволяет задать удобные размеры рабочих окон, используемых при исполнении ПК.

Опции выполнения

Меню Options / Execute (п. 5.1) управляет собственно процессом выполнения, например:

  • останов по первой ошибке,
  • автоматический повтор,
  • тайм‑ауты и др.

Опции протокола

Меню Options / Report (п. 5.2) задаёт детализацию информации, сохраняемой и отображаемой в «Протоколе выполнения». Можно отдельно включать/выключать:

  • комментарии и сообщения оператора,
  • время выполнения команд,
  • расшифровку ошибок,
  • дополнительные диагностические данные.
