2.1.8 Команда Run / Number — найти команду по метке

Горячая клавиша: отсутствует (вызывается только через меню Run → Number).

Назначение

Сервисная команда для быстрого позиционирования в окне «Программа» на команду с заданной меткой. Полезна при больших или разветвлённых программах контроля, где ручной поиск затруднён.

Порядок работы

  1. Откройте меню Run → Number.
  2. В появившемся запросе введите искомую метку команды.
  3. Нажмите Enter:
    • Активным становится окно «Программа»;
    • Курсор переносится на строку начала команды с указанной меткой;
    • Маркер выполнения при этом остаётся на своём месте (текущее состояние выполнения НЕ меняется).

Замечания

  • При ошибке ввода (метка не найдена) курсор остаётся на прежней позиции, выводится звуковой сигнал и/или сообщение об ошибке.
  • Команда служит только для навигации; выполнение программы не запускается автоматически.
  • После позиционирования можно:
    • поставить/снять точку останова (Run / Toggle),
    • сдвинуть маркер выполнения (Run / Goto),
    • или продолжить выполнение (Run / Run, F8 и т. д.).
