3. Сведения о поставке программного обеспечения

3.1 Формат поставки

ПОАСК поставляется вместе с АСК в виде самораспаковывающегося Windows‑архива
mkf_mw.exe на CD/DVD‑диске.

3.2 Содержимое архива

В результате распаковки создаётся стартовая директория MKF_EXE, содержащая:

  1. mkfw.exe — управляющая программа (исполняемый файл);
  2. mkfw.crc — файл для контроля суммы CRC исполняемого кода;
  3. mkf_hrd.cfg — конфигурация аппаратуры конкретной АСК;
  4. папку MKF_HELP — файловую систему справки;
  5. папку EDS_FONT — таблицы кодировки шрифтов и словари (русская орфография, RU ↔ EN).

3.3 Обязательные и необязательные файлы

Для запуска ПОАСК необходим только mkfw.exe. Отсутствие остальных компонентов не вызывает критических ошибок, однако:

  • при отсутствии mkf_hrd.cfg конфигурация аппаратуры будет установлена «по умолчанию» (см. п. 8.2);
  • при отсутствии других требуемых файлов ПОАСК уведомит о невозможности выполнить соответствующую функцию.
