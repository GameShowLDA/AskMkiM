3.1 Подготовка к трансляции

АСК поставляются в различных конфигурациях (например, с разным числом стоек коммутации — СК — и, следовательно, с разным количеством входов). Чтобы корректно транслировать программу контроля, сначала необходимо задать конфигурацию конкретной установки. Это выполняется через меню Options → Config «Конфигурация аппаратуры» (см. [1, п. 8]).

Конфигурация определяется производителем при поставке и сохраняется в файле mki_hrd.cfg в стартовой директории ПО АСК. Если трансляция выполняется на компьютере, который не входит в состав АСК, сначала установите ПО АСК (см. [1, п. 4]) — программа автоматически создаст требуемый файл конфигурации.
