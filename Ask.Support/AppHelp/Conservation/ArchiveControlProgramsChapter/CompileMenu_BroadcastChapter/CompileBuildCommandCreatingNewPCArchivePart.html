2.1 Команда Compile/Build — создание нового архива ПК

Compile → Build «Создать новый архив ПК» запускает мастера построения пустого архива программ контроля. Любое действие, связанное с созданием либо изменением архива, защищено паролем: в текущем сеансе он запрашивается лишь один раз — при первом обращении к архиву.

После ввода пароля появляется стандартное окно выбора файла (см. разд. «[11, п. 10.1]») с заголовком Создать архив ПК. Необходимо указать:

  • путь — существующий и пустой каталог, в котором будет храниться архив;
  • имя файла БДАПК — любое, с расширением .apk (если расширение опустить, .apk подставится автоматически).

Пример: c:\archivpk\archivpk.apk

Если целевой каталог отсутствует, его можно создать прямо из диалога: нажмите TAB, перейдите к списку каталогов и воспользуйтесь F7 «Создать директорию» (см. «[11, п. 10.1.9]»).

После подтверждения пароль‑и‑путь новый архив создаётся и немедленно открывается в режиме редактирования.
