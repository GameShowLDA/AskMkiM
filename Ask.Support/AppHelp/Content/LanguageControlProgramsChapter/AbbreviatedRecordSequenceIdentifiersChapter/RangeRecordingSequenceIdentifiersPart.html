3.1 Диапазонная запись последовательности идентификаторов

3.1.1 Формат диапазонной формы записи

Существует два варианта диапазонной записи

  • Простые идентификаторы:
    Id D Id [ "Шаг" ]
  • Составные идентификаторы:
    Id D Id [ "Шаг" ] S Id [ D Id ] [ "Шаг" ] [ S Id [ D Id ] [ "Шаг" ] … ]

Где:

  • Id — простой идентификатор;
  • Шаг — число (целое или дробное, с "+" или "–") в скобках;
  • D — оператор диапазона: "-", "-*", "-#" или "-&";
  • S — разделитель составного идентификатора: "." (точка), ":" (двоеточие) или "/" (слеш);
  • Квадратные скобки обозначают необязательные части.

Примеры:

  • *X1:1,X1:2,X1:3* ➔ *X1:1-3*
  • *X1:1*X1:2*X1:3* ➔ *X1:1-*3*
  • *X1:1#X1:2#X1:3* ➔ *X1:1-#3*
  • *X1:1#X2:1#X3:1* ➔ *X1-#3:1*

3.1.2 Правила диапазона

Идентификаторы слева и справа от оператора диапазона должны заканчиваться номером (или быть номером). Если шаг не указан, правый номер должен быть больше или равен левому.

Примеры: X11-X13, 1-5.

Для буквенных диапазонов (однобуквенные идентификаторы):

  • Если обе границы «визуально русские» — генерируются русские буквы;
  • Если хотя бы одна «визуально латинская» — генерируются латинские (обе границы при этом должны быть латинскими).

Примеры:

  • А-В или А-b ➔ А, Б, В;
  • А-D ➔ A, B, C, D.

3.1.3 Сокращенная запись

Можно опустить повторный префикс в составных диапазонах.
Пример:
X11-X13:A1-A5 эквивалентно X11-13:A1-5, что разворачивается в X11:A1,…,X11:A5,X12:A1,…,X12:A5,X13:A1,…,X13:A5.

3.1.4 Шаг наращивания

Шаг может быть целым или дробным, положительным или отрицательным.

  • X1:1-11(2) ➔ X1:1,3,5,7,9,11
  • X1:11-1(-2) ➔ X1:11,9,7,5,3,1
  • 1.00-8.50(1.50) ➔ 1.00,2.50,4.00,5.50,7.00,8.50

Если число дробное, длина дробной части в шаге должна совпадать с длиной у границ.
Пример предупреждения: 1-5(0.5) ➔ лучше 1.0-5.0(0.5).
Пример ошибки: 1.00-5.00(1) ➔ нужно 1.00-5.00(1.00).
