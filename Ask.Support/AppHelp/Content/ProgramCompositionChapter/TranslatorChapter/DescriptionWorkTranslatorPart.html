Общее описание

Транслятор — это окно для трансляции и отладки программ контроля. Слева и справа расположены два встроенных редактора, а внизу — список ошибок. Двойной щелчок по ошибке мгновенно подсвечивает соответствующие строки в обоих редакторах и прокручивает их в видимую область.

Из чего состоит окно

Элемент Что делает
Леовое окно - редактор
  • Отображает исходный текст программы контроля.
  • Поддерживает подсветку синтаксиса, перенос и нумерацию строк.
  • Редактирование содержимого: есть.
Правое окно - транслятор
  • Отображает транслированный текст программы контроля.
  • Поддерживает подсветку синтаксиса, нумерацию строк, разделение на блоки кода, выделение зон с ошибками.
  • Редактирование содержимого: нет.
Вертикальный разделитель Позволяет менять ширину левой и правой панелей — потяните мышью за узкую полосу между редакторами.
Список ошибок Таблица внизу с колонками:
  • Код — краткая метка ошибки;
  • Строка — номер строки (обычно исходник → результат);
  • Команда — к чему относится ошибка;
  • Описание — понятное пояснение проблемы.
В заголовке первой колонки отображается общее количество ошибок.
Столбцы можно менять местами - потяните мышью за заголовок стобца и перетащите его.
Горизонтальный разделитель Меняет высоту области ошибок — потяните мышью за тонкую полоску над списком.

Как работать

  1. Откройте файлы: откройте нужный файл через вкладку "Файл".
  2. Выполните проверку/трансляцию: откройте влкадку "Трансляция" и нажмите "Запуск трансляции"
  3. Изучите ошибку: сделайте двойной щелчок по строке в списке — нужные места в обоих окнах подсветятся и окажутся в поле зрения.
  4. Удобнее расположите панели: при необходимости отрегулируйте ширину редакторов и высоту списка ошибок разделителями.

Подсказки

  • Если файл не умещается по ширине, включён перенос строк — горизонтальная прокрутка не нужна.

Что встроено в окна

  • Подсветка синтаксиса (читабельность и удобная навигация по коду/скрипту).
  • Нумерация строк (быстрые ссылки из списка ошибок).
  • Перенос строк (удобнее читать длинные строки).
