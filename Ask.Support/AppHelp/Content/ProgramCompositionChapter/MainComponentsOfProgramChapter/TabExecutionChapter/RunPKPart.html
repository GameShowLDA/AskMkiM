Запуск ПК

Транслирование и запуск кода ПК на аппаратуре (или в холостом режиме).

ВНИМАНИЕ!
Данная функция сработает только при открытом редакторе с ПК. Иначе выдастся сообщение об отсутствии открытого текстовго редактора с ПК.


При запуске программы сначала происходит трансляция кода ПК, затем - запуск на аппаратуре (или в холостом режиме).
Во время запуска появятся два окна:

  • Правое окно - показывает транслированный код ПК с отметкой зелёного цвета. Отметка показывает, на каком этапе сейчас находится ПК;
  • Левое окно - показывает сообщения ПК.
Также с помощью кнопки F5 можно поставить выполнение ПК на паузу, и также снять с паузы.
