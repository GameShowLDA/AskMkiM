Трансляция ПК

Транслирование кода ПК в развёрнутый вид, подготовленный для запуска на аппаратуре.

ВНИМАНИЕ!
Данная функция сработает только при открытом редакторе с ПК. Иначе выдастся сообщение об отсутствии открытого текстовго редактора с ПК.


После выполнения трансляции появятся три окна:

  • Левое окно - оригинальный код ПК;
  • Правое окно - транслированный код ПК, готовый для использования аппаратурой;
  • Нижнее окно - список ошибок и предупреждений оригинального кода ПК.
