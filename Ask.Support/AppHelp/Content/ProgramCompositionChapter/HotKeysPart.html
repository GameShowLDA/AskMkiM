Горячие клавиши

Самые полезные сочетания клавиш. Используйте их для быстрого редактирования и навигации.

Работа с графической оболочкой

КомбинацияДействие
Ctrl + O Функция "Открыть"
Ctrl + N Функция "Создать"
Ctrl + K Функция "Сравнить"
Alt + F4 Выход из программы
Ctrl + S Сохранить изменения в файле
Ctrl + Shift + S Сохранить изменённый файл как
Ctrl + P Печать содержимого редактора

Работа с текстом

КомбинацияДействие
Ctrl + C или Ctrl + Insert Копировать выделение
Ctrl + X или Shift + Delete Вырезать выделение
Ctrl + V или Shift + Insert Вставить скопированный текст
Ctrl + Z Отменить действие
Ctrl + Y Вернуть отменённое действие
Ctrl + A Выделить весь текст
Delete Удалить символ справа
Ctrl + Delete Удалить слово справа
Backspace Удалить символ слева
Ctrl + Backspace Удалить слово слева
Enter Разрыв абзаца (новая строка)
Tab Увеличить отступ выделения
Shift + Tab Уменьшить отступ выделения
Ctrl + D Удалить текущую строку
⇐, ⇑, ⇒, ⇓ Перемещение курсора на символ/строку
Shift + стрелки Расширить выделение на символ/строку
Ctrl + ⇐ или Ctrl + ⇒ Перейти на слово влево/вправо
Ctrl + Shift + ⇐ или Ctrl + Shift + ⇒ Выделить слово влево/вправо
Home или End В начало/конец строки
Shift + Home или Shift + End Выделить до начала/конца строки
Ctrl + Home или Ctrl + End В начало/конец документа
Ctrl + Shift + Home или Ctrl + Shift + End Выделить до начала/конца документа
PageUp или PageDown Переместить курсор на страницу вверх/вниз
Shift + PageUp или Shift + PageDown Выделить страницу вверх/вниз
Alt + Shift + стрелки Прямоугольное выделение посимвольно/построчно

Работа с поиском

КомбинацияДействие
Ctrl + F Открыть панель поиска
F3 или Shift + F3 Найти далее / найти назад
Esc Закрыть панель поиска
Ctrl + H Открыть панель замены

Работа с трансляцией и исполнителем

КомбинацияДействие
F9 Запуск трансляции файла
F8 Поставить точку остановки
Ctrl + F5 Запуск исполнения программы

Дополнительные возможности

КомбинацияДействие
Ctrl + + Увеличить масштаб текста
Ctrl + - Уменьшить масштаб текста
Ctrl + 0 Сбросить масштаб текста
Ctrl + колесо мыши Изменить масштаб текста
Ctrl + M + M Скрыть блок команды (в транслированном файле)
F1 Вызов помощи
навестись мышью на раздел / выделить кодманду + F1 Вызов помощи в конкретном разделе
