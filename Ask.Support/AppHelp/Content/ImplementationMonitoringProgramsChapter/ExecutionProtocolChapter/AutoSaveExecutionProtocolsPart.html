4.3 Автосохранение протоколов выполнения

По завершении работы программы контроля (ПК) текущий протокол выполнения автоматически сохраняется в два каталога, создаваемые в стартовой папке ПО АСК:

  • HISTORY — сохраняет полный протокол «как есть» (п. 2.1.9). Ведение каталога включается/отключается опцией п. 5.2.14.
  • DECINUM — сохраняет протокол в режиме «Только атрибуты ПК и ошибки» (п. 2.1.9). Управляется опцией п. 5.2.15 и поддерживается только Windows‑версией ПО АСК, так как имя каждого файла автоматически содержит децимальный номер ОК и русские символы.

Почему каталог DECINUM не поддерживается в DOS‑версии

  • В MS‑DOS и совместимых системах не поддерживаются «длинные» имена файлов (они усекутся до формата 8.3).
  • В FreeDOS длинные имена допускаются, но русские буквы обрабатываются некорректно: файл становится «невидимым» в других DOS и Windows.
  • Начиная с Windows XP выполнение DOS‑версии ПО АСК в реальном режиме невозможно, а Windows 9x, где это работало, ныне трудно найти.
