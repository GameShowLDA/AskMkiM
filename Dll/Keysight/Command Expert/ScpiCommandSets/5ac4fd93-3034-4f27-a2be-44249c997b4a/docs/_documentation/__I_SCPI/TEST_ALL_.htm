TEST:ALL?

Runs an instrument self-test and returns a pass/fail indication. The TEST:ALL? self-test is more comprehensive than the *TST? self-test.

You must remove all input connections to the instrument before running the full self-test.

Parameter Typical Return
(none) (none)

Run a self-test:

TEST:ALL?

Typical Response: +0 (pass) or +1 (one or more tests failed)
  • If one or more tests fail, the query returns +1 and stores an error in the error queue. For a complete listing of the error messages related to self-test failures, see Self-test Error Messages.
  • Following the test, the instrument returns to the instrument state that existed before the self-test.
