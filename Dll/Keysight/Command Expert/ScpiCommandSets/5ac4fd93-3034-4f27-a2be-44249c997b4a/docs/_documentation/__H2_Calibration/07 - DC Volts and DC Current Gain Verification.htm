DC Volts and DC Current Gain Verification

This procedure verifies DC volts and DC current gain.

  1. Make sure you have read Test Considerations.
  2. Connect the calibrator to the input terminals. For the 34461A/65A/70A, use the front panel input terminals and select the front input terminals with the Front/Rear switch.
  3. Configure each function and range in the order shown in the table below corresponding to the DMM model number. Provide the input shown in the table.
  4. Make a measurement and return the result. Compare measurement results to the test limits shown below. (Be certain to allow for appropriate source settling.)
