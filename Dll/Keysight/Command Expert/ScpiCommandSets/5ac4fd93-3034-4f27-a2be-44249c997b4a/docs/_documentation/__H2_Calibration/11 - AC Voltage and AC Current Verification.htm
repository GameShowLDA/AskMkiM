AC Voltage and AC Current Verification

This procedure verifies AC voltage and current.

  1. Make sure you have read Test Considerations.
  2. Connect the calibrator to the input terminals. For the 34461A/65A/70A, use the front panel input terminals and select the front input terminals with the Front/Rear switch.
  3. Set the AC Volts or AC Current function and the filter as shown below.
  4. Select each range in the order shown below. Provide the indicated input voltage and frequency. Compare measurement results to the test limits shown below. (Be certain to allow for appropriate source settling.)
