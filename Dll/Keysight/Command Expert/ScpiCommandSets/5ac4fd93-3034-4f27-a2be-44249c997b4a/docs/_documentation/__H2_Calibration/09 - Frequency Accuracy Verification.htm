Frequency Accuracy Verification

Configuration: Frequency (CONFigure:FREQuency DEF, MIN)

  1. Make sure you have read Test Considerations.
  2. Select the Frequency function, default range, 3 Hz filter, and 1 second aperture.
  3. Connect the Keysight 33500B Series Function/Arbitrary Waveform Generator to the input terminals. For the 34461A/65A/70A, use the front panel input terminals and select the front input terminals with the Front/Rear switch.
  4. Select each range in the order shown below. Provide the indicated input voltage and frequency. Compare measurement results to the test limits shown below. (Be certain to allow for appropriate source settling.)
