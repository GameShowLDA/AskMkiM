Power-On and Reset State

These tables show factory default settings. Parameters marked with a bullet (•) are nonvolatile and are not affected by power-on or system reset. Other parameters are volatile and are reset to the indicated values at power-on or after *RST or SYSTem:PRESet.

The power-on/reset state may differ from these values if you have enabled power-on state recall mode.
