*LRN?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

*LRN?

Description

This command returns the learn string, which is an ASCII string of SCPI commands. This string includes all of the commands required to put the instrument into its current state.

Return Format

The command returns an ASCII string of SCPI commands, delimited with semicolons (";") between the commands.

Example

The following query returns the learn string.

*LRN?

Typical Response:  A very long string of SCPI commands.
