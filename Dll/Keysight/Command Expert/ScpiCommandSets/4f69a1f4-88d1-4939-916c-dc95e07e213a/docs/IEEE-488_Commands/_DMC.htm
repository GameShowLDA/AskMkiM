*DMC

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

*DMC <string> , [ <block_data>|<macro_string>]

Description

The Define Macro command allows the programmer to assign a sequence of zero or more program message unit elements to a macro label. The sequence is executed when the label is received as a command program header or query program header.

Parameters

Name

Type

Description

Default Value

<string>

String

The macro label.

N/A

<block_data>

Arbitrary Block

The sequence of program message units being labeled.

N/A

<macro_string>

String

The sequence of program message units being labeled.

Examples

*DMC “HOME”,#18MOVE 0,0

The macro descriptively labels a command that sends a pen plotter to its home position.

*DMC “SLOW”,”SPEED SLOW”

The macro defines an abbreviation of a longer command, thus reducing bus traffic.
