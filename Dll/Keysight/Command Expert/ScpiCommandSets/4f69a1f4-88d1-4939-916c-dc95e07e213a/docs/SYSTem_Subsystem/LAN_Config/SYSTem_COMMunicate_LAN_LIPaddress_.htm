SYSTem:COMMunicate:LAN:LIPaddress?

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

SYSTem:COMMunicate:LAN:LIPaddress?

Description

This query returns the "last valid Auto IP address used."

Return Format

The query returns the last valid Auto IP address in the form "nnn.nnn.nnn.nnn" (the quotes are also returned).

Examples

The following query returns the last valid Auto IP address used.

SYST:COMM:LAN:LIP?

Typical Response:  "169.254.20.11"
