Power-On and Reset State

The following tables show the factory defaults for various instrument settings. The parameters marked with a bullet (•) are stored in non-volatile memory, and are not affected by power-on or a system reset. Other parameters are stored in volatile memory and are reset to the indicated values at power-on or after a *RST or SYSTem:PRESet command is issued.

The power-on/reset state may be different than that in the tables if you have enabled the power-on state recall mode. This mode is entered from the Utility menu. See the Agilent 34410A/11A/L4411A User's Guide for further information.
