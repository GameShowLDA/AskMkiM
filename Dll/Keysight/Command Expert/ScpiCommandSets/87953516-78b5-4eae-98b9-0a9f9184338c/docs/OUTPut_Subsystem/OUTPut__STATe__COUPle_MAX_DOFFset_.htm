OUTPut[:STATe]:COUPle:MAX:DOFFset?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

OUTPut[:STATe]:COUPle:MAX:DOFFset?

Description

This query returns the largest delay offset that is required for the output channels installed in this mainframe. Refer to Appendix D in the User's Guide for detailed information.

Return Format

<NR3>

The query command returns the maximum delay offset in the form +1.000000E+01.

Example

The following query returns the maximum delay offset.

OUTP:COUP:MAX:DOFF?

Typical Response:  +3.000000E-02
