MMEMory:EXPort:DLOG

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

MMEMory:EXPort:DLOG <"filename">

Description

This command exports a data log file. The export filename should be the full path and filename. All data is exported in .csv format (comma separated values).

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<"filename">

SPD

Filename and path must be enclosed in single or double quotes.

Required parameter

Example

The following command exports the data log file named data1.dlog to the external USB drive.

MMEM:EXP:DLOG "external:\data1.csv"
