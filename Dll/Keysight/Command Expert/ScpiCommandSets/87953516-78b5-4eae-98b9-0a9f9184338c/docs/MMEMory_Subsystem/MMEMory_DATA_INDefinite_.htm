MMEMory:DATA[:DEFinite]?

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

MMEMory:DATA[:DEFinite]? <"filename">

Description

This query returns the contents of the specified file. The filename should be the full path and filename.

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<"filename">

SPD

Filename and path must be enclosed in single or double quotes.

Required parameter

Return Format

<Block>

The response is a definite length arbitrary binary block of the form:

#<D><DD...D><data-bytes>

#<D> indicates the response is definite length, with <D> the number of digits are in the length field. <DD...D> indicates the length of the binary block containing the actual file. <data-bytes> are the contents of the file.

Examples

The following query returns the contents of a file (quotes are also returned).

MMEM:DATA? "internal:\datalog\data1.dlog"
