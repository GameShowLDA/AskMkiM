[SOURce:]ARB:VOLTage:STAircase:END:TIMe

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

[SOURce:]ARB:VOLTage:STAircase:END:TIMe <time>, (@<chanlist>)

[SOURce:]ARB:VOLTage:STAircase:END:TIMe? [MIN|MAX,] (@<chanlist>)

Description

This command specifies the time in seconds, after the staircase completes, that the end-level voltage persists. Referenced to t2 in the Staircase diagram.

This command is only available on model N6705A with firmware revision A.02.00 or later.

Parameters

Name

Type

Range of Values

Default Value

<time>

NRf+

0 - 262.144  | MIN| MAX

0

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR3>

The query command returns the programmed end time in the form +1.000000E+01. Multiple responses are separated by commas.

Examples

The following command programs an end time of 1 second for the staircase on channel 1.

ARB:VOLT:STA:END:TIM 1,(@1)

The following query returns the end time setting of channel 1.

ARB:VOLT:STA:END:TIM? (@1)

Typical Response:  +1.000000E+00
