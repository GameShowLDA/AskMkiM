[SOURce:]DIGItal:INPut:DATA?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

[SOURce:]DIGital:INPut:DATA?

Description

This query reads the state of the digital control port.  

Return Format

<NR1>

The query returns the binary-weighted value of the state of pins 1 through 7 in bits 0 through 6 respectively.

Example

The following query returns the state of the digital control port pins.

DIG:INP:DATA?

Typical Response:  +0
