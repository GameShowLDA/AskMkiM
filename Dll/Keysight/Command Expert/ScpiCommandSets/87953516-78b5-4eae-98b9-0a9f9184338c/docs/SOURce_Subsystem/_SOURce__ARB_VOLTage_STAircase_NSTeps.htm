[SOURce:]ARB:VOLTage:STAircase:NSTeps

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

[SOURce:]ARB:VOLTage:STAircase:NSTeps <steps>, (@<chanlist>)

[SOURce:]ARB:VOLTage:STAircase:NSTeps? [MIN|MAX,] (@<chanlist>)

Description

This command specifies the number of steps in the staircase.

This command is only available on model N6705A with firmware revision A.02.00 or later.

Parameters

Name

Type

Range of Values

Default Value

<steps>

NRf+

0 - 500 | MIN| MAX

10

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR3>

The query command returns the number of steps in the form +1.000000E+01. Multiple responses are separated by commas.

Examples

The following command programs 10 steps for the staircase on channel 1.

ARB:VOLT:STA:NST 10,(@1)

The following query returns the number of steps of channel 1.

ARB:VOLT:STA:NST? (@1)

Typical Response:  +1.000000E+00
