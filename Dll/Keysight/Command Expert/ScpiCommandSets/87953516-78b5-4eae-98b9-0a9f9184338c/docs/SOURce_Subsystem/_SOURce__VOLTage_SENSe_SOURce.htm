[SOURce:]VOLTage:SENse:SOURce

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

[SOURce:]VOLTage:SENSe:SOURce <state>, (@<chanlist>)

[SOURce:]VOLTage:SENSe:SOURce? (@<chanlist>)

Description

This command sets the state of the remote sense relays.

INTernal

Internal sets the remote sense relays to local sensing. The front panel remote sense terminals are internally connected to the output terminals. The 4 wire indicator is off.

EXTernal

External sets the remote sense relays to remote sensing. The front panel remote sense terminals are enabled and must be connected to the external load. The 4 wire indicator is on.

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<state>

CPD

INTernal | EXTernal

INTernal

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<CRD>

The query command returns INT or EXT.

Example

The following selects EXTernal as the remote sense relay state.

VOLT:SENS:SOUR EXT,(@1)

The following query returns the remote sense relay state.

VOLT:SENS:SOUR? (@1)

Typical Response:  INT
