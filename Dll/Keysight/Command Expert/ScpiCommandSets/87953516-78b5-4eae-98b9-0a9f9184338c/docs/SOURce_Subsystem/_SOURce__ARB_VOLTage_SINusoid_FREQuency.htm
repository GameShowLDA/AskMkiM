[SOURce:]ARB:VOLTage:SINusoid:FREQuency

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

[SOURce:]ARB:VOLTage:SINusoid:FREQuency <frequency>, (@<chanlist>)

[SOURce:]ARB:VOLTage:SINusoid:FREQuency? [MIN|MAX,] (@<chanlist>)

Description

This command specifies the frequency of the sine wave in Hertz.

This command is only available on model N6705A with firmware revision A.02.00 or later.

Parameters

Name

Type

Range of Values

Default Value

<frequency>

NRf+

3.8147E-5 - 1.0E4 | MIN| MAX

1

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR3>

The query command returns the programmed amplitude in the form +1.000000E+01. Multiple responses are separated by commas.

Examples

The following command programs a frequency of 10 Hz on channel 1.

ARB:VOLT:SIN:FREQ 10,(@1)

The following query returns the frequency setting of channel 1.

ARB:VOLT:SIN:FREQ? (@1)

Typical Response:  +1.000000E+01
