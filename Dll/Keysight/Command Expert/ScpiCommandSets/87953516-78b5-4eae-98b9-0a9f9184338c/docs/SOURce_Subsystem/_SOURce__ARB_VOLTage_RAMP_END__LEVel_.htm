[SOURce:]ARB:VOLTage:RAMP:END[:LEVel]

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

[SOURce:]ARB:VOLTage:RAMP:END[:LEVel] <voltage>, (@<chanlist>)

[SOURce:]ARB:VOLTage:RAMP:END[:LEVel]? [MIN|MAX,] (@<chanlist>)

Description

This command specifies the end-level voltage (after the ramp occurs) in volts. Referenced to V1 in the Ramp diagram.

This command is only available on model N6705A with firmware revision A.02.00 or later.

Parameters

Name

Type

Range of Values

Default Value

<voltage>

NRf+

0 - maximum | MIN| MAX

The maximum value is dependent on the voltage rating of the power module.

MIN

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR3>

The query command returns the programmed voltage level in the form +1.000000E+01. Multiple responses are separated by commas.

Examples

The following command programs the end voltage level for the ramp on channel 1.

ARB:VOLT:STEP:END 5,(@1)

The following query returns the end voltage ramp setting of channel 1.

ARB:VOLT:STEP:END? (@1)

Typical Response:  +5.000000E+00
