*RST

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

*RST

Description

This command resets the volatile memory of the instrument to a factory-defined state.  See Factory Reset State for a complete listing of the instrument's factory configuration.

Remarks

  • *RST also forces the ABORt:ACQuire and ABORt:TRANsient commands. This cancels any measurement or output trigger actions presently in process, and resets the two WTG bits in the Status Operation Condition register.

Example

The following command resets the instrument.  

*RST
