Programming Examples

We have included several example programs on the Product Reference CD-ROM shipped along with your Agilent N6700 and N6705A. The examples support the following language environments.

  • Microsoft® Visual Basic 6.0

  • Microsoft® Visual C++ 6.0

  • Microsoft® Excel

  • Agilent VEE Pro

You can also download the examples from the Agilent Web site at www.agilent.com/find/N6700.

For your convenience, this Help document contains several Microsoft Visual Basic 6.0 example programs to help you develop programs for your own application. The example programs are for illustration only, and are provided with the assumption that you are familiar with the programming language being demonstrated and the tools used to create and debug procedures.
