FETCh[:SCALar]:VOLTage[:DC]?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

FETCh[:SCALar]:VOLTage[:DC]? (@<chanlist>)

Description

This query returns the DC output voltage in volts. The data returned by the FETCh command is the result of the last measurement command or acquisition trigger. The data is valid until the next MEASure or INITiate command occurs.

This command is used only with models N6761A/N6762A, or models that have Option 054 (High Speed Test Extensions).

Parameters

Name

Type

Range of Values

Default Value

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Remarks

  • Use this command to return measurement data that has been synchronized with a triggered event.

  • Data is NOT erased from the measurement buffer when it is read by a FETch command.

  • All previous data IS erased from the measurement buffer when a  MEASurement or an INITiate command is executed.

Return Format

<NR3>

The reading is in the form +1.000000E+01 for each channel specified. Multiple responses are separated by commas.

Example

The following query returns the voltage measured on channel 1.

FETC:VOLT? (@1)

Typical Response:  +5.000000E+01
