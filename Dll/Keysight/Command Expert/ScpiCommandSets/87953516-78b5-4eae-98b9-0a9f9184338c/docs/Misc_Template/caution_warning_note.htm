If you prefer creating your own Caution, Warning, and Note entries, here's how:

  1. Create a two-cell table.

  2. Select it by moving your cursor a few pixels off the left end of the table until you see a left-pointing arrow. Then click. (If you select the table by choosing Table > Select Table, the rest of this procedure does not work.)

  3. Choose Format > Borders and Shading.

  4. For Border Width, enter 0 px. Then click OK.

  5. Insert the Caution, Warning, or Note graphic into the left cell in the table.

  6. Drag the divider between the two cells as far to the left as you can.
