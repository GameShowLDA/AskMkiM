SENSe:SWEep:TINTerval

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

SENSe:SWEep:TINTerval <interval>, (@<chanlist>)

SENSe:SWEep:TINTerval? [MIN|MAX,] (@<chanlist>)

Description

This command defines the time period between samples in seconds on models that have measurement controls. Programmed values can range from 20.48 microseconds to 40000 seconds. Values are rounded to the nearest 20.48-microsecond increment.

This command is used only with models N6761A/N6762A, or models that have Option 054 (High Speed Test Extensions).

Parameters

Name

Type

Range of Values

Default Value

<interval>

NRf+

0.00002048 - 40000 | MIN | MAX

20.48 microseconds

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR3>

The query command returns the time interval for each channel specified. The value returned is the time period rounded to the nearest 20.48 microsecond increment; which is not necessarily the value that was programmed. Multiple responses are separated by commas.

Examples

The following command specifies a time interval of 1 second between measurement points on channel 3.

SENS:SWE:TINT 1,(@3)

The following query returns the measurement time interval for channel 1.

SENS:SWE:TINT? (@1)

Typical Response:  +2.048000E-5
