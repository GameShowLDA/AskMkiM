TRIGger:DLOG[:IMMediate]

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

TRIGger:DLOG[:IMMediate]

Description

This command sends an immediate trigger signal to the data logger. This will trigger the data log regardless of the selected data logger trigger source.

This command only applies to the Agilent N6705A DC Power Analyzer.

Remarks

  • You must initiate (run) the data logger before you trigger it.

Example

The following command triggers the data logger.

TRIG:DLOG
