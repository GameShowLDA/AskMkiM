TRIGger:ARB:SOURce

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

TRIGger:ARB:SOURce <source>

TRIGger:ARB:SOURce?

Description

This command selects the trigger source for the arbitrary waveforms. The following trigger sources can be selected:

BUS

Selects a remote interface trigger command.

EXTernal

Selects the rear panel Trigger IN connector.

IMMediate

Triggers the arbitrary waveform immediately.

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<source>

CPD

BUS | EXTernal | IMMediate

IMM

Return Format

<CRD>

The query command returns BUS, EXT, or IMM.

Examples

The following command selects the Trigger IN connector as the arbitrary waveform trigger source.

TRIG:ARB:SOUR EXT

The following query returns the arbitrary waveform trigger source.

TRIG:ARB:SOUR?

Typical Response:  BUS
