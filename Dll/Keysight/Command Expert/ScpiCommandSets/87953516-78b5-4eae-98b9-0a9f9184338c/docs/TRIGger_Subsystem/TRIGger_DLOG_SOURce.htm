TRIGger:DLOG:SOURce

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

TRIGger:DLOG:SOURce <source>

TRIGger:DLOG:SOURce?

Description

This command selects the trigger source for the data logger. The following trigger sources can be selected:

IMMediate

Triggers the data logger immediately.

EXTernal

Selects the rear panel Trigger In connector.

BUS

Selects a remote interface trigger command.

VOLTage<n>

Selects a voltage level. <n> specifies the output channel.

CURRent<n>

Selects a current level. <n> specifies the output channel.

ARSKey

Selects the Arb Run/Stop key.

OOOKey

Selects the Output On/Off keys and the All Outputs On/Off keys.

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<source>

CPD

IMMediate | EXTernal | BUS
| VOLTage<1-4> | CURRent<1-4>
| ARSKey | OOOKey

IMM

Remarks

  • If you have selected a voltage or current level as the trigger source, you must also program the voltage or current level and the polarity. See: TRIGger:DLOG:CURRent, TRIGger:DLOG:CURRent:SLOPe, TRIGger:DLOG:VOLTage, TRIGger:DLOG:VOLTage:SLOPe

Return Format

<CRD>

The query command returns IMM, EXT, BUS, VOLT<1-4>, CURR<1-4>, ARSK, or OOOK.

Examples

The following command selects the Output On/Off key as the data log trigger source.

TRIG:DLOG:SOUR OOOK

The following query returns the data log trigger source.

TRIG:DLOG:SOUR?

Typical Response:  VOLT1
