TRIGger:ACQuire[:IMMediate]

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

TRIGger:ACQuire[:IMMediate] (@<chanlist>)

Description

This command sends an immediate trigger to the measurement system. When the trigger system is initiated, a measurement trigger causes the instrument to measure the output voltage or current, and store the results in a buffer.

This command is used only with models N6761A/N6762A, or models that have Option 054 (High Speed Test Extensions).

Parameters

Name

Type

Range of Values

Default Value

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Remarks

  • You must initiate the measurement trigger system before you can send any triggers.

  • The measured quantity, either voltage or current, must be specified by the SENSe:FUNCtion command.

  • When the triggered measurement has completed, the WTG-meas bit in the Status Operation Condition register is cleared.

Example

The following command generates a measurement trigger on channels 1 and 2.

TRIG:ACQ (@1,2)
