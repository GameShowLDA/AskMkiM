TRIGger:DLOG:CURRent:SLOPe

Syntax | Description | Parameters | Remarks | Return Format | Examples

Syntax

TRIGger:DLOG:CURRent:SLOPe <slope>, (@<chanlist>)

TRIGger:DLOG:CURRent:SLOPe? (@<chanlist>)

Description

This command sets the slope of the current level of the specified output channel when the trigger source for the data logger is set to a current.  

POSitive

Specifies a rising slope of the output signal.

NEGative

Specifies a falling slope of the output signal.

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<slope>

CPD

POSitive | NEGative

POS

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Remarks

  • The output channel specified with this command must match the output channel specified by TRIGger:DLOG:SOURce.

Return Format

<CRD>

The query command returns POS or NEG.

Examples

The following command sets the current slope to Positive on output channel 2.

TRIG:DLOG:CURR:SLOP POS,(@2)

The following query returns the current slope on output channel 2.

TRIG:DLOG:CURR:SLOP (@2)

Typical Response:  POS
