SYSTem:CHANnel:OPTion?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

SYSTem:CHANnel:OPTion? (@<chanlist>)

Description

This query returns a list of options installed in each channel given in the channel list.

Return Format

<SRD>

The query command returns an ASCII string enclosed in double quotes. If no option is installed, an empty quoted string ("") is returned. Multiple responses are separated by commas.

Example

The following query returns the options in channels 2 and 3.

SYST:CHAN:OPT? (@2,3)

Response with options installed in channel 2, none in channel 3: "Opt 054",""
