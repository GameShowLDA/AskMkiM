SYSTem:CHANnel:SERial?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

SYSTem:CHANnel:SERial? (@<chanlist>)

Description

This query returns the serial number of the selected output channel.   

Return Format

<AARD>

The serial number of the output channel. Multiple responses are separated by commas.

Example

The following query returns the serial numbers in channels 2 and 3.

SYST:CHAN:SER? (@2,3)

Typical Response: MY00123456,MY00234567
