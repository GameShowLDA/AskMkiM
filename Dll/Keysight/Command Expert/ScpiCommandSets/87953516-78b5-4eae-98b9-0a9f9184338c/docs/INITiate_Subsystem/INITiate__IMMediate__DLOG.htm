INITiate[:IMMediate]:DLOG

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

INITiate[:IMMediate]:DLOG <"filename">

Description

This command starts a data log. The filename in which to save the data should be the full path and filename. All data log files must have the extension .dlog

This command only applies to the Agilent N6705A DC Power Analyzer.

Parameters

Name

Type

Range of Values

Default Value

<"filename">

SPD

Filename and path must be enclosed in single or double quotes.

Required parameter

Example

The following command starts the data log and creates a file named data1.dlog in the instrument's internal  file system.

INIT:DLOG "internal:\data1.dlog"
