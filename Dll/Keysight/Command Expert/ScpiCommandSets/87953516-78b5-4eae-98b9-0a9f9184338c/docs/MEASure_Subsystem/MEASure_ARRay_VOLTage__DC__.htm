MEASure:ARRay:VOLTage[:DC]?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

MEASure:ARRay:VOLTage[:DC]? (@<chanlist>)

Description

This query initiates and triggers a measurement and returns an array containing the digitized output voltage in volts. The sampling rate is set by SENSe:SWEep:TINTerval. The position of the trigger relative to the beginning of the data buffer is determined by SENSe:SWEep:OFFSet:POINts. The number of points returned is set by SENSe:SWEep:POINts.

This command is used only with models N6761A/N6762A, or models that have Option 054 (High Speed Test Extensions).

Parameters

Name

Type

Range of Values

Default Value

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Remarks

  • Use this command when the measurement does not need to be synchronized with any other event.

  • All previous data is erased from the measurement buffer when a  MEASurement or an INITiate command is executed.

Return Format

<NR3> [,<NR3>]

The reading is in the form +1.000000E+01. Array values are separated by commas.

Example

The following query returns the voltage measured on channel 1.

MEAS:ARR:VOLT? (@1)
