STATus:OPERation:CONDition?

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

STATus:OPERation:CONDition? (@<chanlist>)

Description

This query returns the value of the Operation Status Condition register. The Condition register is a read-only register, which holds the live (unlatched) operational status of the instrument. Reading the Operation Condition register does not clear it.

For more information on the SCPI Status System, see the Status System diagram.

Parameters

Name

Type

Range of Values

Default Value

<chanlist>

Numeric

One or more channels.
(@2) - channel 2
(@1,4) - channels 1 and 4
(@1:3) - channels 1 through 3.

Required parameter

Return Format

<NR1>

The value returned is the binary-weighted sum of all bits set in the register. For example, if bit 3 (decimal value = 8) and bit 4 (decimal value = 16) are set, this command will return 24.

Example

The following command reads the Operation Status Condition register.

STAT:OPER:COND? (@1)

Typical Response:  +24
