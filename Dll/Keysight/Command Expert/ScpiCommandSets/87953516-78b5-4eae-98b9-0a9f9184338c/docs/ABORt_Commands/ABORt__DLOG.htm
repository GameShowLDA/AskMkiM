ABORt:DLOG

Syntax | Description | Parameters | Remarks | Return Format | Example

Syntax

ABORt:DLOG

Description

This command stops the currently running data log and returns the trigger system back to the Idle state. Any remaining data can be fetched until the buffer is emptied.

This command only applies to the Agilent N6705A DC Power Analyzer.

Example

The following command stops the currently running data log.

ABOR:DLOG
