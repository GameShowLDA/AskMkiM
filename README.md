# AskMkiM

**AskMkiM** — многомодульное .NET-решение, предназначенное для разработки, отладки и сопровождения программно-аппаратного комплекса АСК-МКИ (автоматизированная система контроля и измерений).

Проект построен по модульной архитектуре с чётким разделением ответственности между слоями: ядро, движок исполнения, пользовательский интерфейс, служебные библиотеки, тестовые и вспомогательные приложения.

---

## 🧱 Архитектура проекта

Решение включает несколько логически обособленных проектов:

Целевая архитектура и карта миграции зафиксированы в [docs/architecture/clean-architecture.md](docs/architecture/clean-architecture.md).

### 🔹 Core и Engine
- **Ask.Core** — базовые типы, контракты, общие модели и абстракции
- **NewCore** — новое ядро логики (алгоритмы, обработка команд, трансляция)
- **Ask.Engine** — движок выполнения алгоритмов и взаимодействия с оборудованием

### 🔹 UI и взаимодействие с пользователем
- **UI** — основной пользовательский интерфейс
- **MainWindow** — главное окно приложения
- **Message** — подсистема сообщений и вывода информации пользователю

### 🔹 Логирование и поддержка
- **Ask.LogLib** — библиотека логирования
- **Ask.Support** — вспомогательные сервисы и утилиты

### 🔹 Консольные и тестовые приложения
- **ConsoleUI** — консольный интерфейс
- **TestConsole** — тестовое консольное приложение
- **TestWPF** — тестовое WPF-приложение
- **TestManyWindows** — эксперименты с несколькими окнами
- **MethodDependencyExplorer** — инструмент анализа зависимостей методов

### 🔹 Конфигурация и данные
- **DataBaseConfigruration** — работа с конфигурациями и БД
- **Directory.Build.props** — общие настройки сборки для всех проектов
